## POST /matchmaker/extract

Extrai role e skills de um texto ou URL de vaga. Este é o **endpoint legado** — para buscas completas com candidatos, use [/matchmaker/search](/api-reference/agent-system/matchmaker-search).

<Warning>
  Este endpoint **não retorna candidatos**. Ele apenas extrai e normaliza role/skills. Para o fluxo completo de matching, use `/matchmaker/search` ou `/matchmaker/search/stream`.
</Warning>

## Request

```bash
POST /matchmaker/extract
Content-Type: application/json
```

### Body

| Campo | Tipo | Obrigatório | Descrição |
|-------|------|-------------|-----------|
| `query` | string | Sim | Texto ou URL para extração |

## Response

### Schema da Resposta

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `success` | boolean | Se a extração foi bem-sucedida |
| `error_message` | string? | Mensagem de erro |
| `input_type` | string | `"text"` ou `"url"` |
| `original_query` | string | Query original do usuário |
| `scraped_content` | string? | Conteúdo extraído (se URL) |
| `scraped_title` | string? | Título da página (se URL) |
| `llm_extraction` | object | Dados extraídos pela LLM |
| `db_match` | object | Resultado do match na base |
| `db_skills` | array | Skills do role na base |
| `total_skills` | string[] | Merge de todas as skills |

### llm_extraction

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `role` | string? | Cargo extraído pela LLM |
| `seniority` | string? | Senioridade identificada |
| `skills` | string[] | Skills extraídas do texto |

### db_match

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `role` | string? | Nome do role encontrado na base |
| `role_id` | integer? | ID do role na base |
| `match_method` | string? | Método: `"exact"`, `"alias"`, `"trigram"`, `"llm"` |
| `confidence` | float? | Confiança do match (0-1) |
| `seniority` | string? | Senioridade |

## Erros

| Status | Descrição |
|--------|-----------|
| 400 | Query vazia |
| 500 | Erro na extração ou scraping |
