## Visão geral

A extensão `progress/` do Directus expõe dados agregados de desempenho dos jovens: progresso por status de matrícula, proficiência em competências, absorção do conteúdo ao longo do tempo e taxa de conclusão. Todos os dados são lidos das coleções `Matriculas`, `Jovens` e `Turmas`.

### Autenticação

Token Directus válido em `Authorization: Bearer <token>`. Retorna `403` quando não autenticado.

---

## GET /progress/progress/:jovem_id

Retorna a distribuição de matrículas do jovem por status.

### Parâmetros de path

| Parâmetro | Tipo | Descrição |
|---|---|---|
| `jovem_id` | `string` | ID do jovem em `Jovens` |

### Query params

| Parâmetro | Tipo | Descrição |
|---|---|---|
| `percentage` | `boolean` | Se `"true"`, retorna valores como percentual do total |

### Resposta `200`

Status contabilizados: `atrasado`, `concluido`, `nao_iniciado`, `ativo`, `concluido_com_atraso`.

```json
{ "atrasado": 2, "concluido": 8, "nao_iniciado": 1, "ativo": 3 }
```

Com `percentage=true`:

```json
{ "atrasado": 14, "concluido": 57, "nao_iniciado": 7, "ativo": 21 }
```

## GET /progress/proficiencia/:jovem_id

Retorna mediana ou média de proficiência do jovem em competências avaliadas nos pulsos.

### Parâmetros de path

| Parâmetro | Tipo | Descrição |
|---|---|---|
| `jovem_id` | `string` | ID do jovem |

### Query params

| Parâmetro | Tipo | Descrição |
|---|---|---|
| `type` | `string` | `"median"` (default) ou `"average"` |

### Resposta `200`

```json
{ "median": 3.5 }
```

## GET /progress/progress/empresa/:empresa_id

Distribuição agregada de matrículas por status para todos os jovens de uma empresa.

### Parâmetros de path

| Parâmetro | Tipo | Descrição |
|---|---|---|
| `empresa_id` | `string` | ID da empresa |

### Resposta `200`


## GET /progress/absorcao/:empresa_id

Curva de absorção de conteúdo por módulo ao longo do tempo. Mede o quanto do conteúdo de cada módulo foi efetivamente absorvido pelos jovens da empresa.

### Parâmetros de path

| Parâmetro | Tipo | Descrição |
|---|---|---|
| `empresa_id` | `string` | ID da empresa |

### Query params

Suportam arrays JSON serializados (ex.: `[1,2,3]`):

| Parâmetro | Tipo | Descrição |
|---|---|---|
| `trilhas` | `JSON array of numbers` | IDs de trilhas para filtrar |
| `month` | `JSON array of numbers` | Meses para filtrar (1–12) |
| `year` | `JSON array of numbers` | Anos para filtrar |

### Resposta `200`

```json
{
  "data": {
    "Módulo A": [
      { "year": 2026, "entry_month": 3, "value": 72.5 },
      { "year": 2026, "entry_month": 4, "value": 85.0 }
    ]
  },
  "possible_months_to_filter": [3, 4, 5],
  "possible_years_to_filter": [2025, 2026],
  "possible_trilhas_to_filter": [{ "id": 1, "nome": "Trilha Básica" }]
}
```

## GET /progress/conclusao/empresa/:empresa_id

Taxa percentual de conclusão de matrículas para a empresa (jovens ativos).

### Parâmetros de path

| Parâmetro | Tipo | Descrição |
|---|---|---|
| `empresa_id` | `string` | ID da empresa |

### Resposta `200`

```json
{ "percentage": "68" }
```

Retorna `{ "percentage": 0 }` se não houver jovens ativos.

---

## GET /progress/:matricula_id/moodle/integration-data

Retorna dados de integração Moodle para uma matrícula: token de autenticação single-use e username Moodle do jovem.

### Parâmetros de path

| Parâmetro | Tipo | Descrição |
|---|---|---|
| `matricula_id` | `string` | ID da matrícula em `Matriculas` |

### Resposta `200`

```json
{
  "moodle_data": {
    "token": "<md5-hash>",
    "username": "joao.silva",
    "redirect_to": "xpto"
  }
}
```

O token é um hash MD5 de `moodle_id:TOKEN:YYYYMMDD` — válido apenas para o dia atual.
