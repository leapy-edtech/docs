## Autenticação

Todas as APIs utilizam autenticação via **Bearer Token**. Inclua o token no header de cada requisição:

```bash
Authorization: Bearer YOUR_TOKEN_HERE
```

### Obtendo um Token

**Backoffice (Directus)**:
- Tokens de usuário: Faça login via `/auth/login`
- Tokens estáticos: Configure em variáveis de ambiente para integrações

**Agent System**:
- Consulte a documentação específica do Agent System para obter credenciais de API

<Warning>
  Nunca exponha seus tokens em código cliente ou repositórios públicos. Use variáveis de ambiente.
</Warning>

## Rate Limiting

As APIs possuem limites de taxa para garantir estabilidade:

- **Backoffice**: 100 requisições/minuto por IP
- **Agent System**: Consulte documentação específica
