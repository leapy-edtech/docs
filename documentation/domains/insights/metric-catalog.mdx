## Métricas de Engajamento

### eNPS - Employee Net Promoter Score

| Atributo | Valor |
|----------|-------|
| **Fórmula** | % Promotores - % Detratores |
| **Fonte** | Módulo Pulsos |
| **Frequência** | Por ciclo de pulso |
| **Granularidade** | Empresa, Área, Equipe |

### Taxa de Participação em Pulsos

| Atributo | Valor |
|----------|-------|
| **Fórmula** | Respostas / Destinatários × 100 |
| **Fonte** | Módulo Pulsos |
| **Frequência** | Por pulso |
| **Granularidade** | Empresa, Área, Equipe |

### Índice de Engajamento

| Atributo | Valor |
|----------|-------|
| **Fórmula** | Média ponderada das dimensões de clima |
| **Fonte** | Módulo Pulsos |
| **Frequência** | Por ciclo de clima |
| **Granularidade** | Empresa, Área, Equipe |

## Dimensões de Filtro

Todas as métricas podem ser filtradas por:

| Dimensão | Descrição |
|----------|-----------|
| Período | Data/intervalo de tempo |
| Área | Departamento/área |
| Localidade | Unidade/cidade |
| Cargo | Nível/função |
| Tipo de Contrato | CLT, PJ, Estágio |
| Tempo de Casa | Faixas de antiguidade |
