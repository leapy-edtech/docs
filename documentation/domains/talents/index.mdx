## Visão Geral

O módulo de Talentos é a base de dados central de colaboradores na plataforma Leapy. Ele gerencia informações pessoais, profissionais e organizacionais de todos os talentos.

## Informações Gerenciadas

### Dados Pessoais

- Nome completo
- Email e telefone
- Data de nascimento
- Endereço

### Dados Profissionais

- Cargo e função
- Área e equipe
- Data de admissão
- Tipo de contrato
- Gestor direto

### Competências

- Skills técnicas
- Competências comportamentais
- Certificações
- Idiomas

### Histórico

- Movimentações de cargo
- Avaliações de desempenho
- Treinamentos realizados
- Participação em projetos

## Ciclo de Vida do Talento

```mermaid
stateDiagram-v2
    [*] --> PreOnboarding : Contratação
    PreOnboarding --> Active : Início
    Active --> OnLeave : Afastamento
    OnLeave --> Active : Retorno
    Active --> Offboarding : Desligamento
    Offboarding --> Inactive : Finalizado
    Inactive --> [*]
```
