## Jobs de CriaĂ§ĂŁo

### 1. `createPulsosJovens`

**PropĂłsito**: Criar pulsos para todos os jovens ativos conforme agendamento.

**Trigger**: Cron diĂĄrio Ă s 9h

**LĂłgica**:

```typescript

   - AND enabled = true
   - AND next_pulso_date <= TODAY

   a. Buscar jovens ativos da empresa
   b. Para cada jovem:
```

## Troubleshooting

### Job Falhou

1. Acesse Inngest Dashboard > Functions > Selecione o job
2. Veja os **Runs** recentes
3. Abra o run falhado
4. Analise o erro em cada **Step**
5. Corrija o problema (dados, lĂłgica, integraĂ§ĂŁo)
6. Re-execute manualmente se necessĂĄrio

### Job Travado

Se job estĂĄ "Running" por > 5min:

1. Cancele manualmente no Inngest
2. Investigue causa (query lenta? API externa?)

## Jobs de AtualizaĂ§ĂŁo de Status de Pulsos Jovens

### 17. `updateStatusPulsosJovensParaPendentes`

**ID:** `update-pulsos-pendentes`

**PropĂłsito:** Atualiza o status dos pulsos de jovens selecionados para `"pendente"`. Executa em lotes paralelos para evitar sobrecarga no banco.

**Evento ouvido:** `backoffice/pulsos-jovens.update-to-pendente`

**Payload:**

```json
{ "ids": [1, 2, 3, 45] }
```

**LĂłgica:**
1. Divide o array de IDs em chunks
2. Para cada chunk, executa `updateDirectusData("pulsos_jovens", chunk, { status: "pendente" })`
3. Todos os chunks rodam em paralelo com `Promise.all`
4. Retorna os registros atualizados com campos `id`, `data_aplicacao`, `status`

**Arquivo:** `src/inngest/functions/update-pulsos-jovens-to-pendentes.ts`

---

### 18. `updateStatusPulsosJovensParaVencidos`

**ID:** `update-pulsos-vencidos`

**PropĂłsito:** Atualiza o status dos pulsos de jovens selecionados para `"vencido"`. Mesma lĂłgica de chunking do job anterior.

**Evento ouvido:** `backoffice/pulsos-jovens.update-to-vencidos`

**Payload:**

```json
{ "ids": [1, 2, 3] }
```

**LĂłgica:**
1. Divide IDs em chunks
2. Atualiza `status = "vencido"` em paralelo via `updateDirectusData`
3. Retorna registros atualizados com `id`, `data_aplicacao`, `status`

**Arquivo:** `src/inngest/functions/update-pulsos-jovens-to-vencidos.ts`
