## Visão Geral

O módulo de Cursos e Conteúdos oferece uma plataforma completa de aprendizagem para desenvolvimento dos colaboradores, com cursos, trilhas e conteúdos educacionais.

## Tipos de Conteúdo

| Tipo | Descrição |
|------|-----------|
| **Curso** | Conteúdo estruturado com módulos e avaliações |
| **Trilha** | Sequência de cursos para desenvolver competências |
| **Vídeo** | Conteúdo em formato de vídeo |
| **Artigo** | Texto educacional ou informativo |
| **Podcast** | Conteúdo em áudio |
| **Material** | PDFs, apresentações, documentos |

## Jornada de Aprendizagem

flowchart LR
    A[Descobrir] --> B[Inscrever]

    D --> E[Certificar]

## Integrações

O módulo se integra com:

- **Talentos**: Competências desenvolvidas
- **Matchmaker**: Requisitos de oportunidades
- **Presenças**: Eventos presenciais de treinamento
- **Insights**: Métricas de desenvolvimento
