title: "Glossário"
description: "Terminologia e conceitos utilizados na plataforma Leapy"
---

## Termos Gerais

| Termo | Definição |
|-------|-----------|
| **Tenant** | Organização/empresa cliente que utiliza a plataforma |
| **Talento** | Colaborador cadastrado na plataforma |
| **RH** | Usuário com perfil de Recursos Humanos |
| **Liderança** | Usuário com perfil de gestor/líder |

## Pulsos

| Termo | Definição |
|-------|-----------|
| **Pulso** | Pesquisa ou questionário enviado aos talentos |
| **Ciclo** | Período de aplicação de um pulso |
| **Resposta** | Feedback coletado de um talento |
| **NPS** | Net Promoter Score - métrica de satisfação |
| **eNPS** | Employee Net Promoter Score - NPS aplicado a colaboradores |

## Matchmaker

| Termo | Definição |
|-------|-----------|
| **Match** | Conexão sugerida entre talento e oportunidade |
| **Oportunidade** | Vaga, projeto ou posição disponível |
| **Score** | Pontuação de compatibilidade entre perfis |
| **Interesse** | Manifestação de interesse em uma oportunidade |

## Insights

| Termo | Definição |
|-------|-----------|
| **Dashboard** | Painel visual com métricas e indicadores |
| **KPI** | Key Performance Indicator - indicador chave de desempenho |
| **Drill-down** | Navegação em níveis de detalhe dos dados |
| **Slice** | Recorte ou filtro aplicado aos dados |

## Admin

| Termo | Definição |
|-------|-----------|
| **Role** | Papel ou função de um usuário no sistema |
| **Permission** | Permissão específica de acesso ou ação |
| **Scope** | Escopo de visibilidade de dados |
| **Audit Log** | Registro de auditoria de ações |

## Plataforma

| Termo | Definição |
|-------|-----------|
| **Backoffice** | Sistema administrativo (Directus) |
| **Job** | Tarefa assíncrona processada em background |
| **Event** | Evento do sistema que dispara ações |
| **Webhook** | Notificação HTTP enviada para sistemas externos |
| **Flow** | Fluxo automatizado no Directus |
