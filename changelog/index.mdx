## 2026

### Maio 2026

#### [v0.1.0.1](https://github.com/leapy-edtech/leapy/releases/tag/v0.1.0.1) — 2026-05-05

**Planejamento**

- **Plano de migração revisado:** Backoffice/Users se torna a Fase 1 da migração pro monorepo, substituindo Pulsos/Talentos. Blast radius ~20x menor (5-15 usuários internos vs 200+ externos), ciclo de feedback mais rápido e validação do padrão de Vercel rewrite reusado em todas as fases subsequentes.
- Pulsos/Talentos realocado para Fase 4, sem mudança de escopo.
- Task 0.11 adicionada: Vercel rewrite + hardening CORS no leapy-rh como pré-requisito da Fase 1.
- Docs de migração reorganizados para refletir a nova numeração de fases.

## 2024

### Janeiro 2024

#### v2.5.0 - 15 de Janeiro

**✨ Novas Funcionalidades**

- **Agent System**: Novo sistema de assistentes de IA
  - HR Assistant para consultas de dados
  - Support Agent para diagnóstico de problemas
  - Analytics Agent para insights

- **Matchmaker**: Melhorias no algoritmo de matching
  - Score de compatibilidade mais preciso
  - Novas dimensões de análise
  - Sugestões em tempo real

**🐛 Correções**

- Corrigido erro de timeout em exportações grandes
- Ajustado cálculo de eNPS para cenários edge-case
- Melhorada performance de dashboards com muitos dados

**🔧 Melhorias**

- Interface de pulsos redesenhada
- Novos filtros em dashboards
- Otimização de queries de relatórios
