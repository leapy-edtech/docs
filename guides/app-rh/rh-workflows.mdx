## Criar e Aplicar Pulso

### Passo a Passo

1. **Acessar Pulsos**
   - Menu lateral > Pulsos > Criar Novo

2. **Configurar Pulso**
   - Escolher template ou criar do zero
   - Definir título e descrição
   - Adicionar perguntas

3. **Definir Público**
   - Toda a organização
   - Áreas específicas
   - Lista manual de talentos

4. **Configurar Datas**
   - Data de início
   - Data de encerramento
   - Lembretes automáticos

5. **Revisar e Ativar**
   - Visualizar prévia
   - Confirmar configurações
   - Ativar pulso

### Acompanhar Respostas

Durante o período de coleta:

- Dashboard de progresso em tempo real
- Taxa de resposta por área
- Envio de lembretes manuais
- Prorrogação de prazo (se necessário)

### Analisar Resultados

Após encerramento:

- Score geral e por dimensão
- Comparativo com períodos anteriores
- Breakdown por área/equipe
- Exportação de relatórios

## Cadastrar Novo Talento

### Individual

1. **Acessar Talentos > Novo**
2. **Preencher dados obrigatórios**
   - Nome completo
   - Email
   - Data de admissão
   - Área e cargo
3. **Preencher dados opcionais**
   - Telefone
   - Gestor direto
   - Dados complementares
4. **Salvar**
   - Convite de acesso é enviado automaticamente

### Importação em Massa

1. **Baixar template CSV**
2. **Preencher dados**
3. **Fazer upload**
4. **Validar mapeamento**
5. **Confirmar importação**

## Criar Oportunidade

1. **Acessar Matchmaker > Nova Oportunidade**
2. **Preencher informações**
   - Título e descrição
   - Tipo (vaga, projeto, etc)
   - Área e gestor responsável
3. **Definir requisitos**
   - Competências necessárias
   - Experiência
   - Disponibilidade
4. **Configurar visibilidade**
   - Quem pode ver e se candidatar
5. **Publicar**

## Gerenciar Candidaturas

### Triagem

1. Acessar candidaturas da oportunidade
2. Revisar perfil do candidato
3. Verificar score de compatibilidade
4. Avançar ou recusar

### Processo Seletivo

1. Agendar entrevistas
2. Registrar feedback
3. Tomar decisão
4. Comunicar resultado

### Finalização

1. Confirmar seleção
2. Solicitar aprovação do gestor atual (se necessário)
3. Definir data de transição
4. Comunicar todas as partes
